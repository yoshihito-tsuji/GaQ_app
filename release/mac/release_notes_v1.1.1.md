## 主な変更内容

### Mac版
- Python 3.12.3環境で動作確認済み
- すべての主要機能が正常動作
- ドラッグ&ドロップでのファイル選択対応
- リアルタイム進捗表示
- クリップボードコピー機能

- **SHA256**: 下記ファイルで確認

## 動作環境

### macOS版
- macOS 10.15以降
- 推奨: 8GB RAM以上
- オフライン動作（初回起動時のみモデルダウンロードが必要）

## インストール方法

1. DMGファイルをダウンロード
2. DMGをダブルクリックしてマウント
3. "GaQ Offline Transcriber.app" を Applications フォルダにドラッグ
4. Applications フォルダから起動
