# GaQ Offline Transcriber v1.1.0

## 概要
オフラインで動作するAI文字起こしアプリケーション

## 特徴
- faster-whisper（高精度AI音声認識）を使用
- 初回起動時に音声認識モデル（約1.5GB）をダウンロード
- モデルダウンロード後は完全にオフラインで動作
- プライバシー保護：音声データは外部に送信されません

## 動作環境
- macOS 10.15 (Catalina) 以降
- Python環境同梱（追加インストール不要）
- 初回のみインターネット接続必須

## ダウンロード
[DMGファイルへのリンク]

## 使用方法
1. DMGファイルをダウンロード
2. GaQ Offline Transcriber.appをApplicationsフォルダにドラッグ
3. 初回起動時に音声認識モデルを自動ダウンロード
4. 音声ファイルをアップロードして文字起こし実行
