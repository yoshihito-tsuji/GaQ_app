## サンプル音声ファイルの要件

### 基本要件

| 項目 | 要件 |
|------|------|
| **形式** | M4A、MP3、WAV（iOSがサポートする形式） |
| **長さ** | **10分前後**（8-12分でOK） |
| **言語** | **日本語** |
| **品質** | 明瞭な発音、ノイズ少ない |
| **サンプルレート** | 44.1kHz または 48kHz |
| **ビットレート** | 128kbps以上（音質優先の場合は256kbps） |
| **ファイルサイズ** | 10-30MB程度 |

### 内容の推奨事項

**適した内容**:
- ✅ 講演・プレゼンテーション（話者1名、明瞭な発音）
- ✅ ニュース・ドキュメンタリー（標準語、ナレーション）
- ✅ 教育コンテンツ（授業、解説動画）
- ✅ インタビュー（話者2-3名、交互に発話）

**避けるべき内容**:
- ❌ 音楽（歌詞の文字起こしは不正確）
- ❌ 方言が強い会話（標準語以外の精度が低下）
- ❌ 複数人が同時に話す会議（音声分離が困難）
- ❌ ノイズが多い環境音（街中、工事現場など）

## 利用上の注意

### 1. 著作権・権利クリア

**重要**: サンプル音声は以下のいずれかを使用してください。

#### 推奨オプション

**A. 自分で録音する**
- ✅ 権利問題なし
- ✅ 内容を自由に設定可能
- 方法: iPhoneの「ボイスメモ」アプリで録音

**B. パブリックドメイン・CC0音源**
- ✅ 商用利用可能
- ✅ 著作権フリー
- 入手先: 後述の「推奨サンプル音声の取得方法」を参照

**C. 許諾済みコンテンツ**
- ✅ 権利者の許可を得た音声
- 例: 研究室の講演録音（辻先生の許可を得たもの）

#### 禁止事項

- ❌ **市販の音声コンテンツ**（オーディオブック、ポッドキャストの無断使用）
- ❌ **YouTubeの無断ダウンロード**（私的利用でも著作権法違反の可能性）
- ❌ **テレビ番組の録音**（放送局の権利侵害）

### 2. 外部公開の禁止

**注意**: このディレクトリのサンプル音声は、**外部公開しません**。

**理由**:
- プライバシー保護（話者の同意が必要）
- 著作権リスク回避
- リポジトリサイズの肥大化防止

**GitHubへの公開範囲**:
- ✅ README.md（このファイル）のみ公開
- ❌ 音声ファイル本体は非公開（`.gitignore`で除外済み）

### 3. テスト専用利用

**用途**:
- ✅ 性能計測（Phase 1）
- ✅ 開発中の動作確認（Phase 2-3）
- ✅ プライベートなテスト

**禁止**:
- ❌ 公開デモでの使用（権利クリアが必要）
- ❌ 第三者への配布
- ❌ App Store提出時のスクリーンショット素材（プライバシー配慮）

## 推奨サンプル音声の取得方法

### 方法1: 自分で録音（最も簡単・安全）

#### iPhoneの「ボイスメモ」を使用

**手順**:
1. iPhoneで「**ボイスメモ**」アプリを起動
2. 赤い録音ボタンをタップ
3. 以下のいずれかを録音:
   - 自分で10分間話す（テーマ: 好きな話題、プロジェクト説明など）
   - ニュース・YouTubeの音声を再生しながら録音（**テスト目的のみ、非公開厳守**）
4. 停止ボタンをタップ
5. ファイル名を「sample_10min」に変更
6. 共有ボタン → 「ファイルに保存」→ Macに転送

**録音のコツ**:
- 静かな部屋で録音
- iPhoneを口元から20-30cm離す
- ゆっくり、明瞭に話す

## ファイルの配置方法

### 手順1: 音声ファイルの準備

1. 上記いずれかの方法で10分音声を取得
2. ファイル名を`sample_10min.m4a`（またはMP3/WAV）にリネーム

### 手順2: このディレクトリに配置

**Macでの操作**:
```bash
# 現在のディレクトリに移動
cd /Users/yoshihitotsuji/Claude_Code/GaQ_app/docs/ios_phase1/sample_audio

# ダウンロードフォルダから音声ファイルをコピー
cp ~/Downloads/sample_10min.m4a .

# ファイルが配置されたことを確認
ls -lh
```

**期待される出力**:
```
total 20M
-rw-r--r--  1 yoshihito  staff   15M Oct 23 14:30 sample_10min.m4a
-rw-r--r--  1 yoshihito  staff  5.2K Oct 23 14:00 README.md
```

### 手順3: iPhoneに転送（実機テスト用）

**方法A: Finderを使用**（推奨）

1. iPhoneをMacに接続
2. Finderでサイドバーから「iPhone」を選択
3. 「ファイル」タブをクリック
4. WhisperKitSampleアプリ（または作成したアプリ名）を選択
5. `sample_10min.m4a`をドラッグ&ドロップ

**方法B: AirDropを使用**

1. Macで音声ファイルを右クリック
2. 「共有」→「AirDrop」
3. iPhoneを選択
4. iPhone側で「受け入れる」
